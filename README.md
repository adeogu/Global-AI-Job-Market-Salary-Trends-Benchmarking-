# Global-AI-Job-Market-Salary-Trends-Benchmarking-  
# Setup Instructions

1- Clone the repository

2- Open Anaconda Prompt

3- Navigate to the directory where you cloned the repository

4- Activate the dmml environment:

conda activate dmml


5- Install the required dependencies:

pip install -r requirements.txt


6- Run the Application/Start the app:

python app.py

7- After it starts, copy the HTTP address shown in the terminal and paste it into your browser to access the application.
